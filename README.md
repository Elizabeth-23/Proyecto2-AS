# Proyecto2-AS
Proyecto 2 - Arquitectura de Software

Software Web de Control Escolar que permite registrar alumnos con los siguientes campos:   

  - Matricula: String (10 caracteres) 
  - Nombre completo  String  
  - Promedio  double (Mayor que 0 y menor que 100)   

También crear una entidad profesor:   

  - Identificador: String (10 caracteres) 
  - Nombre completo: String  
  - Horas  int (mayor o igual a 0)   

Y crear una entidad de relación llamada Tutorías:

  - Identificador del Profesor: String (10 caracteres) 
  - Matrícula del Alumno: String (10 caracteres) 
  - Día de tutoría: String (Lunes a Viernes)
